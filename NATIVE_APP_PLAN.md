# RiffTrax Notifier — Native macOS Menu Bar App

## What We're Building

A native macOS app that:
- Lives in the **menu bar** (top-right area of your screen, like Spotify, Dropbox, etc.)
- Has no Dock icon — it's purely a background utility
- Can be **enabled/disabled** with a toggle in its menu
- Shows the current movie title and trivia in a **floating desktop overlay** (replacing Übersicht)
- Fires **system notifications** when a new movie starts
- Replaces the three separate scripts (`bot.py`, `trivia_watcher.py`) and the Übersicht widget with one self-contained app

## Mental Model: How a macOS Menu Bar App Works

Before writing any code, here's the conceptual map:

| Concept | What it is | Replaces |
|---|---|---|
| `MenuBarExtra` | The icon + popover in the top menu bar | Nothing — this is new |
| `NowPlayingStore` | Shared state object the whole app reads | The `.txt` files on disk |
| `IRCConnection` | Background task reading Twitch chat | `bot.py` |
| `TriviaService` | Calls Claude API when title changes | `trivia_watcher.py` |
| `OverlayWindow` | Transparent floating window on the desktop | Übersicht widget |
| `UserNotifications` | Native macOS notifications | `osascript` in bot.py |

The `.txt` files (`~/.rifftrax_now_playing.txt`, etc.) are no longer needed — state lives in memory inside the app. The app reads/writes them only if you want to keep Übersicht running alongside during a transition period.

## Step 2: Configure the App as a Menu Bar App (No Dock Icon)

Open `Info.plist`. If you don't see it, click your project name in the left panel → select the target → **Info** tab.

Add this key:

```
Application is agent (UIElement) = YES
```

In raw XML (`Info.plist` source), this looks like:
```xml
<key>LSUIElement</key>
<true/>
```

Also add notification permission description:
```xml
<key>NSUserNotificationsUsageDescription</key>
<string>RiffTrax Notifier sends alerts when a new movie starts.</string>
```

**What this does**: `LSUIElement = YES` tells macOS this app has no Dock presence and no default menu bar (File, Edit, etc.). It becomes a pure background utility — exactly like 1Password mini, Bartender, etc.

## Step 4: The Shared State Object (`NowPlayingStore.swift`)

This is the heart of the app. Everything reads from and writes to this one object.

```swift
import Foundation
import Observation

@Observable
final class NowPlayingStore {
    // Current movie title. Empty string = nothing playing.
    var movieTitle: String = ""

    // Trivia text generated by Claude. Empty = not yet loaded.
    var trivia: String = ""

    // Whether the bot is actively watching Twitch chat.
    var isEnabled: Bool = true {
        didSet {
            UserDefaults.standard.set(isEnabled, forKey: "isEnabled")
        }
    }

    // True while trivia is being fetched from the API.
    var isFetchingTrivia: Bool = false

    init() {
        // Restore the on/off toggle state across app launches
        self.isEnabled = UserDefaults.standard.object(forKey: "isEnabled") as? Bool ?? true
    }

    // Called when a new movie is confirmed (either from admin command or sync)
    func updateMovie(title: String) {
        guard title != movieTitle else { return }
        movieTitle = title
        trivia = ""  // Clear old trivia — TriviaService will refill it
    }

    func clearMovie() {
        movieTitle = ""
        trivia = ""
    }
}
```

**Key concept — `@Observable`**: This macro (macOS 14+) makes the class "reactive." Whenever a property changes, any SwiftUI view watching it automatically redraws. It's the glue between your background IRC work and your UI.

## Step 5: The IRC Connection (`IRCConnection.swift`)

This replaces `bot.py`. It opens a raw TCP connection to Twitch's IRC server and reads lines.

```swift
import Foundation
import Network

final class IRCConnection {
    private var connection: NWConnection?
    private var buffer = ""
    private let store: NowPlayingStore
    private let notificationManager: NotificationManager

    // Same constants as bot.py
    private let host = "irc.chat.twitch.tv"
    private let port: UInt16 = 6667
    private let nick = "justinfan70"
    private let channel = "rifftrax"
    private let adminTrigger = "!cmd edit movie"
    private let streamElementsBot = "streamelements"
    private let rifftraxPrefix = "RiffTrax"
    private let movieQueryCommands: Set<String> = ["!movie", "!film"]
    private let queryWindowSeconds: Double = 30

    private var lastQueryTime: Date? = nil
    private var reconnectTask: Task<Void, Never>?

    init(store: NowPlayingStore, notificationManager: NotificationManager) {
        self.store = store
        self.notificationManager = notificationManager
    }

    func start() {
        connect()
    }

    func stop() {
        reconnectTask?.cancel()
        connection?.cancel()
        connection = nil
    }

    private func connect() {
        let endpoint = NWEndpoint.hostPort(
            host: NWEndpoint.Host(host),
            port: NWEndpoint.Port(rawValue: port)!
        )
        connection = NWConnection(to: endpoint, using: .tcp)

        connection?.stateUpdateHandler = { [weak self] state in
            switch state {
            case .ready:
                self?.sendHandshake()
                self?.receiveNextLine()
            case .failed(let error):
                print("[irc] Connection failed: \(error). Reconnecting in 5s...")
                self?.scheduleReconnect()
            case .cancelled:
                break
            default:
                break
            }
        }

        connection?.start(queue: .global(qos: .background))
    }

    private func scheduleReconnect() {
        reconnectTask = Task {
            try? await Task.sleep(for: .seconds(5))
            guard !Task.isCancelled else { return }
            connect()
        }
    }

    private func sendHandshake() {
        send("NICK \(nick)")
        send("JOIN #\(channel)")
    }

    private func send(_ text: String) {
        let data = (text + "\r\n").data(using: .utf8)!
        connection?.send(content: data, completion: .idempotent)
    }

    // Read data continuously, split into lines, process each
    private func receiveNextLine() {
        connection?.receive(minimumIncompleteLength: 1, maximumLength: 4096) { [weak self] data, _, isComplete, error in
            guard let self else { return }

            if let data, let text = String(data: data, encoding: .utf8) {
                buffer += text
                let lines = buffer.components(separatedBy: "\r\n")
                // Last element may be incomplete — keep it in the buffer
                buffer = lines.last ?? ""
                for line in lines.dropLast() where !line.isEmpty {
                    self.handleLine(line)
                }
            }

            if error == nil && !isComplete {
                self.receiveNextLine()
            }
        }
    }

    private func handleLine(_ line: String) {
        // Respond to server keepalive pings
        if line.hasPrefix("PING") {
            send("PONG :tmi.twitch.tv")
            return
        }

        guard let (username, message) = parsePRIVMSG(line) else { return }
        print("[\(username)]: \(message)")

        if message.hasPrefix(adminTrigger) {
            // Admin changed the movie
            let title = extractTitle(from: message)
            Task { @MainActor in
                store.updateMovie(title: title)
                notificationManager.sendNotification(subtitle: "Now Starting", body: title)
            }

        } else if movieQueryCommands.contains(message.trimmingCharacters(in: .whitespaces).lowercased()) {
            // A viewer asked !movie or !film — start the window
            lastQueryTime = Date()

        } else if username == streamElementsBot && message.hasPrefix(rifftraxPrefix) {
            // StreamElements replied — only act if preceded by !movie/!film within window
            if let queryTime = lastQueryTime,
               Date().timeIntervalSince(queryTime) <= queryWindowSeconds {
                lastQueryTime = nil
                let title = stripURL(from: message)
                Task { @MainActor in
                    if !title.isEmpty && title != store.movieTitle {
                        print("[irc] SYNC: missed a change, updating to: \(title)")
                        store.updateMovie(title: title)
                        notificationManager.sendNotification(subtitle: "Now Playing", body: title)
                    }
                }
            }
        }
    }

    // Parse ":nick!nick@host PRIVMSG #channel :text" → (nick, text)
    private func parsePRIVMSG(_ line: String) -> (String, String)? {
        guard line.contains("PRIVMSG") else { return nil }
        let parts = line.components(separatedBy: " PRIVMSG ")
        guard parts.count >= 2 else { return nil }
        let username = parts[0].trimmingCharacters(in: .init(charactersIn: ":")).components(separatedBy: "!").first ?? ""
        let messageStart = parts[1].components(separatedBy: ":").dropFirst().joined(separator: ":")
        return (username, messageStart.trimmingCharacters(in: .whitespacesAndNewlines))
    }

    private func extractTitle(from message: String) -> String {
        let rest = String(message.dropFirst(adminTrigger.count)).trimmingCharacters(in: .whitespaces)
        return stripURL(from: rest)
    }

    private func stripURL(from text: String) -> String {
        let words = text.split(separator: " ").filter { !$0.hasPrefix("http") }
        return words.joined(separator: " ").trimmingCharacters(in: .whitespaces)
    }
}
```

## Step 6: Trivia Service (`TriviaService.swift`)

Replaces `trivia_watcher.py`. Watches the store and calls Claude when the title changes.

The Anthropic API key is read from the same file as before: `~/.rifftrax_anthropic_key`.

```swift
import Foundation

final class TriviaService {
    private let store: NowPlayingStore
    private var watchTask: Task<Void, Never>?
    private var lastFetchedTitle = ""

    private let model = "claude-haiku-4-5-20251001"
    private let fallback = "Even our robots couldn't find anything nice to say about this one."

    init(store: NowPlayingStore) {
        self.store = store
    }

    func start() {
        watchTask = Task { [weak self] in
            // Poll the store's title every 2 seconds
            // (In a fancier version you'd use withObservationTracking — this is simpler to understand)
            while !Task.isCancelled {
                try? await Task.sleep(for: .seconds(2))
                await self?.checkForTitleChange()
            }
        }
    }

    func stop() {
        watchTask?.cancel()
    }

    @MainActor
    private func checkForTitleChange() async {
        let title = store.movieTitle

        if title == lastFetchedTitle { return }
        lastFetchedTitle = title

        if title.isEmpty {
            store.trivia = ""
            return
        }

        store.isFetchingTrivia = true
        let result = await fetchTrivia(for: title)
        // Only apply if the title hasn't changed while we were waiting
        if store.movieTitle == title {
            store.trivia = result
        }
        store.isFetchingTrivia = false
    }

    private func fetchTrivia(for title: String) async -> String {
        guard let apiKey = loadAPIKey() else {
            print("[trivia] No API key found at ~/.rifftrax_anthropic_key")
            return fallback
        }

        let prompt = """
        You are a writer for RiffTrax, the comedy riffing show. Write 2-3 short, punchy \
        sentences of trivia about "\(title)" in the RiffTrax voice — dry wit, affectionate \
        mockery, genuine facts wrapped in a joke. If you don't recognize the title, write \
        something generically funny about low-budget cinema. No bullet points, no lists, \
        just flowing sentences.
        """

        var request = URLRequest(url: URL(string: "https://api.anthropic.com/v1/messages")!)
        request.httpMethod = "POST"
        request.setValue("application/json", forHTTPHeaderField: "Content-Type")
        request.setValue(apiKey, forHTTPHeaderField: "x-api-key")
        request.setValue("2023-06-01", forHTTPHeaderField: "anthropic-version")
        request.httpBody = try? JSONSerialization.data(withJSONObject: [
            "model": model,
            "max_tokens": 200,
            "messages": [["role": "user", "content": prompt]]
        ])

        do {
            let (data, _) = try await URLSession.shared.data(for: request)
            if let json = try? JSONSerialization.jsonObject(with: data) as? [String: Any],
               let content = json["content"] as? [[String: Any]],
               let text = content.first?["text"] as? String {
                return text.trimmingCharacters(in: .whitespacesAndNewlines)
            }
        } catch {
            print("[trivia] API error: \(error)")
        }

        return fallback
    }

    private func loadAPIKey() -> String? {
        let path = (("~/.rifftrax_anthropic_key") as NSString).expandingTildeInPath
        return try? String(contentsOfFile: path, encoding: .utf8).trimmingCharacters(in: .whitespacesAndNewlines)
    }
}
```

## Step 7: Notifications (`NotificationManager.swift`)

Replaces the `osascript` calls in `bot.py`. Uses the native UserNotifications framework — these are the same style of notification as Mail, Calendar, etc.

```swift
import UserNotifications

final class NotificationManager {
    func requestPermission() {
        UNUserNotificationCenter.current().requestAuthorization(options: [.alert, .sound]) { granted, error in
            if !granted {
                print("[notifications] Permission denied. Enable in System Settings → Notifications.")
            }
        }
    }

    func sendNotification(subtitle: String, body: String) {
        let content = UNMutableNotificationContent()
        content.title = "RiffTrax"
        content.subtitle = subtitle
        content.body = body
        content.sound = .default

        let request = UNNotificationRequest(
            identifier: UUID().uuidString,
            content: content,
            trigger: nil  // nil = deliver immediately
        )

        UNUserNotificationCenter.current().add(request)
    }
}
```

## Step 9: The Desktop Overlay (`OverlayWindow.swift`)

This replaces the Übersicht widget. It's a transparent, click-through window that floats on the desktop.

```swift
import AppKit
import SwiftUI

final class OverlayWindow: NSWindow {
    init(store: NowPlayingStore) {
        // Create a borderless, transparent window
        super.init(
            contentRect: NSRect(x: 40, y: 0, width: 525, height: 200),
            styleMask: [.borderless],
            backing: .buffered,
            defer: false
        )

        // Float above the desktop but below normal windows (like Übersicht)
        self.level = .init(rawValue: Int(CGWindowLevelForKey(.desktopIconWindow)) + 1)

        // Transparent background
        self.backgroundColor = .clear
        self.isOpaque = false

        // Don't steal focus when clicked through; ignore mouse events
        self.ignoresMouseEvents = true
        self.collectionBehavior = [.canJoinAllSpaces, .stationary, .ignoresCycle]

        // Position: 40pt from top-left (matching Übersicht widget)
        if let screen = NSScreen.main {
            let y = screen.frame.maxY - 40 - 150  // 40pt from top, 150pt height estimate
            self.setFrameOrigin(NSPoint(x: 40, y: y))
        }

        // Embed SwiftUI content
        let hostingView = NSHostingView(rootView: OverlayView(store: store))
        hostingView.frame = self.contentView!.bounds
        hostingView.autoresizingMask = [.width, .height]
        self.contentView = hostingView
    }
}

struct OverlayView: View {
    var store: NowPlayingStore
    @State private var triviaOffset: CGFloat = 0

    var body: some View {
        Group {
            if !store.movieTitle.isEmpty {
                VStack(alignment: .leading, spacing: 0) {
                    // Title card
                    VStack(alignment: .leading, spacing: 6) {
                        Text("📽 Now Riffing")
                            .font(.system(size: 13, weight: .semibold))
                            .tracking(1.5)
                            .textCase(.uppercase)
                            .foregroundColor(Color(red: 0.9, green: 0.04, blue: 0.08))

                        Text(store.movieTitle)
                            .font(.system(size: 22, weight: .semibold))
                            .foregroundColor(.white)
                            .lineLimit(3)
                    }
                    .padding(.horizontal, 22)
                    .padding(.vertical, 15)
                    .background(Color.black.opacity(0.75))
                    .overlay(
                        Rectangle()
                            .frame(width: 4)
                            .foregroundColor(Color(red: 0.9, green: 0.04, blue: 0.08)),
                        alignment: .leading
                    )
                    .cornerRadius(4)

                    // Trivia ticker (only if available)
                    if !store.trivia.isEmpty {
                        Divider().background(Color.white.opacity(0.15))

                        ScrollingTickerView(text: store.trivia)
                            .frame(height: 24)
                            .background(Color.black.opacity(0.75))
                    }
                }
                .frame(maxWidth: 525, alignment: .leading)
            }
        }
        .frame(maxWidth: .infinity, maxHeight: .infinity, alignment: .topLeading)
        .padding(40)
    }
}

// Scrolling marquee, matching the Übersicht ticker behaviour
struct ScrollingTickerView: View {
    let text: String
    @State private var offset: CGFloat = 0
    @State private var textWidth: CGFloat = 0

    var body: some View {
        GeometryReader { geo in
            Text("🎬  " + text + "     ")
                .font(.system(size: 13))
                .foregroundColor(Color.white.opacity(0.7))
                .fixedSize()
                .background(
                    GeometryReader { inner in
                        Color.clear.onAppear { textWidth = inner.size.width }
                    }
                )
                .offset(x: offset)
                .onAppear {
                    offset = geo.size.width
                    withAnimation(.linear(duration: Double(textWidth) / 40).repeatForever(autoreverses: false)) {
                        offset = -textWidth
                    }
                }
                .onChange(of: text) {
                    // Reset animation when trivia changes
                    offset = geo.size.width
                    withAnimation(.linear(duration: Double(textWidth) / 40).repeatForever(autoreverses: false)) {
                        offset = -textWidth
                    }
                }
        }
        .clipped()
        .padding(.horizontal, 22)
    }
}
```

## Step 10: Wire Everything Together (`RiffTraxNotifierApp.swift`)

This is the app entry point — the `@main` file that Xcode generated. Replace its contents entirely:

```swift
import SwiftUI
import AppKit

@main
struct RiffTraxNotifierApp: App {
    @State private var store = NowPlayingStore()
    private let notificationManager = NotificationManager()
    private var ircConnection: IRCConnection?
    private var triviaService: TriviaService?
    private var overlayWindow: OverlayWindow?

    init() {
        notificationManager.requestPermission()

        // Create the floating overlay window
        let overlay = OverlayWindow(store: store)
        overlay.orderFront(nil)
        self.overlayWindow = overlay

        // Start background services
        let irc = IRCConnection(store: store, notificationManager: notificationManager)
        let trivia = TriviaService(store: store)
        self.ircConnection = irc
        self.triviaService = trivia
        irc.start()
        trivia.start()
    }

    var body: some Scene {
        MenuBarExtra {
            MenuBarView(store: store)
        } label: {
            // Show a filled icon when monitoring, outline when disabled
            Image(systemName: store.isEnabled ? "popcorn.fill" : "popcorn")
        }
        .menuBarExtraStyle(.window)  // Shows a popover panel, not a plain text menu
    }
}
```

**What `.menuBarExtraStyle(.window)` does**: Your MenuBarView renders as a floating panel when clicked, not a list of menu items. This gives you the richer layout with toggles and text.

## Step 11: Handle the On/Off Toggle

When `store.isEnabled` changes, the IRC connection needs to start or stop. Add an `onChange` modifier.

In `RiffTraxNotifierApp.swift`, replace the `MenuBarExtra` body:

```swift
var body: some Scene {
    MenuBarExtra {
        MenuBarView(store: store)
            .onChange(of: store.isEnabled) { _, enabled in
                if enabled {
                    ircConnection?.start()
                } else {
                    ircConnection?.stop()
                    store.clearMovie()
                }
            }
    } label: {
        Image(systemName: store.isEnabled ? "popcorn.fill" : "popcorn")
    }
    .menuBarExtraStyle(.window)
}
```

## Suggested Build Order

Build and test in this sequence — each step is independently testable:

1. **App + MenuBarExtra** — get the icon in the menu bar with a hard-coded title string
2. **NowPlayingStore + MenuBarView** — wire up the toggle and display the hard-coded store state
3. **OverlayWindow** — confirm it appears over the desktop with the hard-coded title
4. **IRCConnection** — connect to Twitch, print lines to Xcode console (no state changes yet)
5. **Wire IRC → Store** — title updates flow to overlay and menu bar
6. **NotificationManager** — notifications fire on title change
7. **TriviaService** — trivia fetches and appears in overlay/popover
8. **On/Off toggle** — enable/disable IRC connection cleanly
9. **Login item** — register as startup item

This order means you always have a working app at each step and can test each piece in isolation.

## Xcode Tips for Beginners

- **⌘R** — build and run
- **⌘B** — build without running (good for checking errors)
- **⌘.** — stop the running app
- The **Debug console** at the bottom shows your `print()` output — use it like Python's terminal
- Red error markers appear inline in code; click them for the full message
- If something looks gray/unavailable in a menu, make sure the project (not a file) is selected in the left panel
- **⌘Z** works as undo in code files just like anywhere else
